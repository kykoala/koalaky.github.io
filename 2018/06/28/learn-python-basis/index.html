python3的基础套餐

  • 2018-06-28

为什么要学python? 因为知道pyhton的人很多，但是真正会的人却很少！

迈入python的大门

安装python3.6

python官网传送门 =====> PYTHON走你~
下载、安装、配置环境变量，然后win+R cmd 打开命令行后输入python -V 查看当前版本号(注意是大写的V)
OK 输入python就进入了python的鱼骨头环境咯~

基础语法

基础数据类型

数字(Number)
  • 整数(int)
    • python2:L在2版本，用来标识长整数的 Long
    • python3:没有长短一分
  • 浮点数(float)
    • 小数有精度
  • 布尔(Bool) 大小写敏感
    • #NameError: name ‘true’ is not defined
    • True
    • False
  • 复数(complex)
    • 2+3j
    • 虚部以大J，小j结尾的都是复数
字符串(str)
  • 表示方法以下都可以
    • ‘abc’
    • “abc”
    • ‘’’abc’’’
    • “””abc”””
  • 索引: str[index]使用index选择访问位置，索引从左向右、从0开始，从右向左、从-1开始。
  • 切片: str[start:stop:[step]],切片可以获取字符串某个区域的值，这个值还是字符串类型。另外字符串中的内容不可变，So字符串为不可变数据对象哦~

小练习:

列表(list)
  • 表示方式: [‘a’,’b’]
  • 列表也可以使用索引和切片操作
  • 列表中的数据可以被修改，列表为可变数据对象

小练习:

元祖(tuple)
  • 表示方法: (‘a’,’b’)
  • 元祖也可以使用索引和切片操作
  • 元祖中的数据不可以被修改，元祖为不可变数据对象
  • 如果只创建含有一个元素的元祖会被解释器认为是一个实际数据对象，不会解释为元祖
    举个栗子:
    • (a) ===> 这是一个数据对象
    • (a,) ===> 这才是一个元祖

小练习:

变量本质

1、变量不需要先定义再使用，可以直接给变量赋值
2、变量可以重复存储不同的数据类型
3、可以同时为多个变量赋值
4、变量名可以由字母、数字、下划线构成，数字不可以作为开头，大小写敏感，起名字最好是见名知意英文
5、不支持自增和自减（++ ,- -）

变量的引用计数

python中相同数据的赋值会共享同一片空间地址，并不是占用一个新的地址，所以为了记录当前使用这个地址的变量有多少，就有了引用计数的概念。这样做的好处在于可以节约内存，防止产生多余的内存碎片。

  • 我们可以使用sys模块下的getrefcount()函数进行变量引用计数的查看
