好用的小工具，小玩意儿

Anywhere 随启随用的静态文件服务器

还在为没有服务的本地静态资源无法让手机访问嘛，anywhere可以解决！用法很easy!
来到npm的官网后，搜索anywhere 找到anywhere这个插件 用npm isntall去安装就好了！
npm install anywhere -g
Anywhere传送门~
安装好了之后直接anywhere -p 8000就可以访问了！不需要服务器的哦！ 这里的-p是你本机的ip，8000是端口号。官网也有对应的用法~
