从零学习 -- Python模块篇

  • 2018-07-26

Python本身就内置了很多非常有用的模块，只要安装完毕，这些模块就可以立刻使用。python中需要用import导入。如果是第三方的模块则需要先用pip进行下载安装。

Python的内置模块

OS模块

os模块是干什么的呢？

1、与当前操作系统进行交互
2、删除，创建文件及文件夹、获取目录，获取文件夹结构等

1、os.system函数可以执行或者打开对应操作系统下的程序根据你输入的命令

2、os.name 输出当前os模块在哪个平台下

posix：类Unix操作 安卓 Centos ubuntu Debian
nt：Win
mac：MacOS
dos：DOS

3、os.getcwd() -> str 获取当前的程序工作目录

4、os.listdir(path=os.getcwd()) -> list 返回指定path目录下的所有文件及文件夹的列表

  • 返回的结果是一个字符串列表，并且每一个路径都是相对路径的，另外该函数也会将隐藏文件展示。

5、os.remove(file_path)

  • 删除file_path所指定的一个文件
  • 如果成功返回None，失败了直接报错
  • 删除的时候一定要指明文件的后缀

6、os.rmdir(dir_path)

  • 删除dir_path指定的一个文件夹(目录)
  • 如果成功返回None，失败了直接报错
  • 无法递归删除文件夹，如果文件夹不为空，那么删除不了

7、os.makedirs(‘a/b/c’)

  • 递归创建目录
  • 可以嵌套创建目录
  • 成功返回None，失败则报错

8、os.mkdir(‘a’)

  • 创建目录
  • 不能在linux下创建一个和文件名重名的文件夹，虽然类型不同，但是名字也不能是相同的
  • 成功返回None，失败则报错

9、os.chdir()

  • 在程序运行期间可以通过该函数来改变工作目录

10、os.rename()

  • os模块重命名文件

os模块下的子模块

os.path可以处理和路径及文件类型有关的问题

1、os.path.isfile(path) -> Bool

  • 判断路径是否是一个真正的文件

2、os.path.isdir(path) -> Bool

  • 判断路径是否是一个真正的目录

3、os.path.join(‘path’,’sub_path’)

  • 拼接path和sub_path
  • 可以构成绝对路径
  • 路径中重复的部分不会被去掉

4、os.path.exists(path)

  • 判断路径是否存在
  • 如果存在返回True，反之返回False

5、os.path.getsize(path)

  • 返回路径对应的文件大小
  • 返回的文件大小单位为Byte
文件操作

文件操作，在程序里都是对内存中的文件进行操作！
1、创建文件
2、打开文件

  • path: 需要打开文件的路径
  • mode: 打开文件所使用的权限

读：r read
提供读操作
单纯的读权限打开不存在文件，会直接报错
写：w write
提供写操作
追加：a append
类似与写权限，只提供对于文件的写操作
在权限参数的传递过程中，如果只是单纯的给了一个权限，那么其他权限不享有
如果我读写兼备，那么只需要在权限补充一个+号
r+
w+
a+

  • open函数返回的是一个文件的句柄

3、读文件

  • 函数直接阅读整个文件，并返回整个文件为一个字符串
  • 把整个文件读到了内存
  • 如果给read函数传递了num变量，那么他是通过num来读取对应文件中的指定字符个数
  • read函数获取到的文件内容会保留行末尾的\n\r
  • readline函数一次可以读取文件中的一行内容
  • 当遇到了\n\r则停止，代表当前是一行了
  • 当遇到了EOF（文件结束标志），readline如果去读取超过文件本身行数的次数，那么不会报错，只会返回空
  • readline不会一次性把整个文件展开在内存，但是需要我们手动的维护行数
  • readline函数在获取到文件内容的每一行后，也会保留结尾的\r\n
  • 返回了字符串列表
  • 列表中每一个元素都是一行的数据
  • 并且保留行末尾的换行符号

4、写文件

  • 成功写入缓冲区中的字符数，用来向文件写入str
  • write函数在写入文件内容的时候，不会自动的带换行\r\n
  • 成功时返回None，反之则报错
  • 用来向文件写入字符串列表，而且列表中的每一个字符串数据不会自动加换行

Python中自动维护了一个叫做读写指针的东西，这个读写指针会自动的根据你访问文件的量来进行向后移动，读写公用同一个指针。

5、关闭文件

6、获取文件读写指针位置

  • 获取文件读写指针的位置
  • 返回值获取到的是读写指针的字节数位置

7、读写指针修改

  • Python中文件读写指针共享

    offset：偏移量
    字节为单位
    when: 从哪里开始移动
    0:文件开头
    1:当前位置
    2:文件末尾
    a模式下的文件末尾读写指针如何移动到文件头部呢？
    fp.seek(0,0) -> 回到了文件开头

时间模块

time模块处理精确的秒数较多

  • 时间获取

  • 获取精确秒数

  • 时间字符串展示转换

1、time.time()

  • 这里获取的是一个浮点数类型时间戳（linux时间：从1970年1月1日， 00:00:00）

2、time.localtime(float_time)

  • 如果不传递参数，那么默认返回本地时间的一个时间元组

    res = time.localtime() -> 时间结果是具备时区的
    time.struct_time(
    tm_year=2018, 年
    tm_mon=7, 月
    tm_mday=31, 日
    tm_hour=14, 时
    tm_min=36, 分
    tm_sec=33, 秒
    tm_wday=1, 星期几 星期一数字是0 从0开始 表示星期几
    tm_yday=212, 今年第几天
    tm_isdst=0 夏令时
    )

3、time.sleep(seconds)

  • 程序休眠，让程序挂起，seconds参数是一个秒数

4、time.clock()

  • 一般衡量程序的耗时

    win：
    第一次调用：返回当前的CPU时间
    第二次调用：返回距离上一次调用所花费的时间
    linux：
    直接返回程序运行到执行这个函数所花费的时间

5、time.asctime(时间元组)

  • 处理时间元组成为一个可读性更好的字符串

6、time.strftime(格式，时间元组) -> 良好可读性的字符串

  • 将时间元组根据你指定的格式来成为一个良好可读性的字符串
    python中时间日期格式化符号：

    %y 两位数的年份表示（00-99）
    %Y 四位数的年份表示（000-9999）
    %m 月份（01-12）
    %d 月内中的一天（0-31）
    %H 24小时制小时数（0-23）
    %I 12小时制小时数（01-12）
    %M 分钟数（00=59）
    %S 秒（00-59）
    %a 本地简化星期名称
    %A 本地完整星期名称
    %b 本地简化的月份名称
    %B 本地完整的月份名称
    %c 本地相应的日期表示和时间表示
    %j 年内的一天（001-366）
    %p 本地A.M.或P.M.的等价符
    %U 一年中的星期数（00-53）星期天为星期的开始
    %w 星期（0-6），星期天为星期的开始
    %W 一年中的星期数（00-53）星期一为星期的开始
    %x 本地相应的日期表示
    %X 本地相应的时间表示
    %Z 当前时区的名称
    %% %号本身
    年月日，时分秒中，只有月和天是小写的，其他都是大写的

7、time.strptime(良好可读性的字符串，格式) -> 时间元组

  • 把一个字符串变成时间元组。再变回去的时候，要和之前变化的时候一一对应。
datetime模块

  • 处理年月日，时分秒的时间创建

  • 时间差值计算

  • 时间获取

1、datetime.datetime

  • 处理年月日，时分秒的

  • 获取当前的时间

  • 返回值：datetime.datetime类型
  • 返回值类型支持差值运算，用来求出两个时间区间的秒数，或者说你指定的时间单位

2、datetime.timedelta()

  • timedalte 是datetime中的一个对象，该对象表示两个时间的差值
  • 其中参数都是可选，默认值为0
  • timedalte 有三个只读属性：
    • timedelta.min：负数最大时间差，相当于 timedelta(-999999999)。
    • timedelta.max：正数最大时间差，相当于 timedelta(days=999999999, hours=23, minutes=59, seconds=59, microseconds=999999)。
    • timedelta.resolution：两个时间的最小差值 相当于 timedelta(microseconds=1)。

3、datetime.date

  • 处理年月日，返回当前时间
    datetime.date.today()

4、datetime.time()

  • 只能处理时分秒，多用来创建时分秒的时间

创建时间
datetime.datetime(2018,6,6,10,10,59) 创建年月日，时分秒
datetime.date(2018,6,6) 创建年月日
datetime.time(10,50,59) 创建时分秒

random模块

顾名思义，这个模块就是用来产生随机数的。那么我们来看一下这个模块的使用方法，上代码咯！
1、random.randrange(stop)

  • 生产出从0到stop区间内的一个随机整数，不包含stop（即 [0, stop) ）

注意，这里的stop也是一个整数哦！不然就会抛出异常咯（ValueError: non-integer arg 1 for randrange()）

2、random.randint(start,stop)

返回start和stop区间内的一个随机整数，起点和终点都可能被取到，这里的start和stop也都是整数哦!

3、random.getrandbits(num)

  • 传入一个num值，0-2**num次方，去一个随机的整数
  • 如果传递的num是1，0 - 2 ** 1
  • 左闭右开 [0, 2 ** num)

4、random.random()
返回介于0到1之间的浮点数

5、random.uniform(start,stop)

  • 取出一个从start开始到stop结束的一个随机浮点数

  • 左闭右开 [start,stop)

  • 这里的start和stop可以是整数也可以是浮点数

6、random.choice(seq)

  • 从一个非空序列中随机选择一个元素
  • 序列为空则报错
  • 序列：字符串，列表，元组 支持索引操作的数据

7、random.shuffle(seq)

  • 打乱序列的顺序

8、random.sample(seq,num)

  • num是我要从seq中随机抽取数据的长度，num一定要小于len(seq)，否则会抛出异常（ValueError: Sample larger than population or is negative）
  • 从seq中 随机抽取num个数据 返回成一个列表
