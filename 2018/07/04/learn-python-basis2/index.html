字典

字典的概念
  • 这里的mydict就是一个字典，字典是以键值对的形式（{key: value}）,或许你会觉得这个形式很眼熟，emmm = =~，好像json啊！！嗯，样子看起来是差不多，但是这两个东东可不是一回事儿哦！
  • 字典中的key值会进行hash运算生成hash表，所以字典的访问效率要优于普通序列数据类型(列表、元祖)
  • value值没有限制，可以使任意数据类型，但是有一些可变数据类型不可以做key值
  • 字典是可变数据类型

  • 获取所有values值: dict.values()

用dict[key]这种方法我们可以更新字典，并且如果字典中的key存在，则更新key对应的value值，如果key不存在，那么就会将key和其对应的value追加到字典中。

集合

集合的概念
  • 把不同元素组合在一起形成的一个数据集合
  • 集合分为可变集合和不可变集合
创建集合

1、大括号

访问集合
更新集合
  • set.add(): 更新可hash数据类型到集合中,并维持原样
  • set.update(): 如果传入的是一个序列，将会被拆分成单个元素，并且去重。update不接收一个不可拆分的数据，比如整数.
  • 可变集合支持更新操作。不可变集合不支持更新操作
删除集合中的元素

remove方法可以删除我们想要删除的集合中的元素，但是如果删除一个不存在的值的时候会报错。

discard方法可以删除我们想要删除的集合中的元素，但是如果删除一个不存在的值的时候不会报错。

del myset 会将我们的整个元素都删掉
