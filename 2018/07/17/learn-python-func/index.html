函数

为什么要用到函数
  • 按照过程编写代码，一般功能都是一次性的，非常不好维护
  • 把功能封装集成，方便二次开发和维护
  • def 语句 定义函数的语句
函数的参数

  • 形参 是为了方便在提前编写函数时，可以先在函数内部使用，挖个坑

    • 形参你可以想成是一个待赋值的变量

  • 实参 就是函数实际调用时传递进去的值

参数探究

  • 必备参数

    • 当你定义函数时，形参已经有了，在实参传递的过程，一定要和形参一一对应

  • 命名参数

    *当函数参数太多了，记不住顺序呀 可以指定的写明形参名去赋值

  • 缺省参数

    • 可以在函数定义时，指明一个形参的默认值，就可以不给这个具有默认值的形参传递实参了

    • 当你给一个具有默认参数的形参传递一个实参，那么会覆盖掉原有的默认值

    • 缺省参数的定义顺序一定是从右向左，从后向前

  • 不定长参数

    • 元组不定长 *arg

      • *号 只是在定义形参时做标识的

    • 字典不定长 **kwargs

      • 这种不定长传参一定要 命名不定长

    • 联合使用

      • 不定长参数可以接收空值

返回值
  • 函数调用完成之后会返回结果
  • 默认的函数如果没有返回值，那么返回None

递归函数

一个函数如果直接或间接地调用了自身,那么这个函数就是递归函数。

  • 这里的func就是一个递归函数，在函数体内又调用了func它自身。
  • 递归有最大的上限数，为1000次左右。
  • 每一次执行递归函数都会开启一个新的函数空间，每次递归就是执行一次函数调用，而函数的调用就是进栈和出栈的一系列操作，执行的过多如果没有及时的释放资源，那么你会看到你的资源管理器中的内存使用率会不断的飙升！（Bomm~）
  • 所以使用的时候切记切记要添加退出递归的条件！
